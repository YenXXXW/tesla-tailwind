## **About the project**
A fully mobile responsive project created with **NextJS**.

## Technologies used
* NextJS
* Framer Motion
* Tailwind CSS


## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
